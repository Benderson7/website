My Projects

Lucid

Summer 2020 (Unity 3D Engine, C#)

Lucid is a 3D turn-based roguelike video game that I worked on in my Game Programming course. The game has an emphasis on randomized runthroughs and different kinds of fighting styles that come together during the rhythm-like combat. My largest contributions are creating the level-generator, designing the enemy AI, and setting up the animations and battle-scenes.

TimeLost

Summer 2020 (Java)

"TimeLost" is an original tabletop roleplaying game that I run with my friends. Since the system is unique, I created this GUI application, which helps them keep track of their character's stats and make actions.
